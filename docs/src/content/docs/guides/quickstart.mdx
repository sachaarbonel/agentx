title: Quickstart

Run the included example to see Glass Hands in action.

## Requirements

- Rust (stable, edition 2021)
- Chrome or Chromium on your PATH
- `OPENAI_API_KEY` set

## Run

```bash
export OPENAI_API_KEY=sk-...
RUST_LOG=info cargo run --example quickstart
```

The example opens Chromium and executes a small task using the OpenAI Responses API with computer-use.
